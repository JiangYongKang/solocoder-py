# Doubly Linked List (双向链表)

## 模块功能

本模块提供了一个完整的双向链表（Doubly Linked List）数据结构实现。每个节点同时持有前驱（prev）和后继（next）两个指针，支持：

- 在任意位置（头部、尾部、指定节点之后）插入节点
- 删除任意位置的节点
- 从头到尾的正向遍历和从尾到头的反向遍历
- 反转整个链表
- 判断链表是否为空、获取当前节点数量

## 核心类的职责

### `Node`
链表节点类，使用 `@dataclass` 装饰器定义。

| 属性   | 类型              | 说明                       |
|--------|-------------------|----------------------------|
| `data` | `Any`             | 节点存储的数据，支持任意类型 |
| `prev` | `Optional[Node]`  | 前驱节点指针               |
| `next` | `Optional[Node]`  | 后继节点指针               |

### `DoublyLinkedList`
双向链表主类，封装了所有链表操作。

| 属性     | 类型              | 说明                     |
|----------|-------------------|--------------------------|
| `head`   | `Optional[Node]`  | 只读，链表头节点          |
| `tail`   | `Optional[Node]`  | 只读，链表尾节点          |
| `size`   | `int`             | 只读，当前节点数量        |

### 异常类

| 异常类                     | 基类                   | 触发场景                                   |
|---------------------------|------------------------|-------------------------------------------|
| `DoublyLinkedListError`   | `Exception`            | 所有链表异常的基类                          |
| `NodeNotFoundError`       | `DoublyLinkedListError`| 在 `insert_after` 中传入 `None` 或不属于该链表的节点 |

## 操作的时间复杂度

| 操作                         | 时间复杂度   | 说明                                                   |
|------------------------------|-------------|-------------------------------------------------------|
| `prepend(data)`              | O(1)        | 在头部插入，直接操作 `head` 指针                         |
| `append(data)`               | O(1)        | 在尾部插入，直接操作 `tail`
