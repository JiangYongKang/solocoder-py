在当前 Python 项目中实现一个栈（Stack）数据结构，使用 Python 内置的列表或数组作为底层存储。
1. 实现栈的基本操作：压入元素到栈顶、弹出栈顶元素并返回、查看栈顶元素但不移除。
2. 实现栈的查询操作：判断栈是否为空、获取当前栈中的元素数量。
3. 边界处理：当从空栈弹出元素或查看栈顶时，应抛出明确的异常而不是返回 None 或产生未定义行为。
4. 代码实现放在 src/solocoder_py/stack/ 包下，为该模块编写完整的单元测试，覆盖正常流程（顺序压入弹出、查看栈顶、判空、获取大小）和边界条件（空栈弹出抛异常、空栈查看栈顶抛异常、大量元素压入后逐一弹出、压入弹出交替执行）。测试代码放在 tests/stack/ 下。
