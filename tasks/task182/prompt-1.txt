在当前 Python 项目中实现指数加权移动平均（EWMA），使用内存数据结构维护在线计算状态。
1. 实现带平滑因子 α 的 EWMA 计算：EWMA 的递推公式为 S_t = α * x_t + (1 - α) * S_{t-1}，其中 α 为平滑因子（0 < α ≤ 1），x_t 为当前值，S_t 为当前指数加权均值。α 在初始化时由调用方指定。提供查询当前 EWMA 值的方法。
2. 实现预热期限制：在数据点不足时（即早期样本数量较少时），EWMA 值受初始值影响过大，需要支持预热期校正。预热期内使用偏差校正公式 S_t_corrected = S_t / (1 - (1-α)^t)，当 t 足够大使得校正因子接近 1 时自动退出预热模式。预热期的窗口长度可配置。
3. 实现 NaN 安全重置：当输入值 x_t 为 NaN 或 Infinity 时，EWMA 计算器应能安全处理。遇到 NaN 时，跳过当前值不更新状态（或重置为上一个有效值），遇到 Inf 时应特殊标记并阻止后续计算被污染。提供 reset 方法将状态重置为初始值，方便异常恢复后重新开始计算。
4. 代码实现需要放在 src/solocoder_py/ewma/ 包下，然后为该模块编写完整的单元测试，覆盖正常流程（固定序列的 EWMA 值与手动递推计算一致、偏差校正后预热期内值与理论值吻合、预热期后自动退出校正模式、reset 后重新计算正确）、边界条件（α 接近 0 时的极端平滑行为、α=1 时等价于当前值直通、仅一个数据点时的行为、α 取边界值的处理）和异常分支（α 超出 (0,1] 范围时构造拒绝、NaN 输入后的状态保持或跳过、Inf 输入的处理与后续计算的隔离、预热期长度超出合理范围的校验）。测试代码放在 tests/ewma/ 下。
5. 在该模块包内编写 README.md，说明模块功能、核心类的职责、EWMA 递推公式与偏差校正推导、预热期策略、NaN 安全机制以及使用示例。
