在当前 Python 项目中实现一致性哈希环域，所有相关代码放在 src/solocoder_py/consistent_hash/ 包下，使用内存数据结构维护节点与虚拟节点。
1. 实现哈希环节点管理：支持添加和删除物理节点，每个物理节点可以配置虚拟节点数量。虚拟节点需要映射回所属物理节点。
2. 实现按 key 路由查询：给定任意 key，返回哈希环上顺时针方向命中的物理节点。空环查询需要返回明确错误。
3. 实现最小迁移统计：节点新增或删除后，可以比较变更前后的 key 分布，统计被迁移的 key 数量和迁移比例，用于验证一致性哈希的迁移范围。
4. 支持节点权重：不同节点可以配置不同权重，权重越高应拥有更多虚拟节点，并在大量 key 分布时体现更高的命中比例。
5. 支持环状态查询：可以查询当前物理节点列表、虚拟节点数量、每个节点负责的 key 数量估算和哈希环快照。
6. 为该模块编写完整的单元测试，覆盖正常流程（添加节点、删除节点、按 key 路由）、边界条件（空环、单节点、重复添加节点）和异常分支（删除不存在节点、非法虚拟节点数量、权重非法、节点变更后的迁移比例校验）。测试代码放在 tests/consistent_hash/ 下。
7. 在该模块包内编写 README.md，说明模块功能、核心类的职责、哈希环路由规则与节点迁移示例。
