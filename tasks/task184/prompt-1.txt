在当前 Python 项目中实现简单线性回归器，使用内存数据结构维护在线学习的模型参数。
1. 实现在线梯度下降更新参数：回归器维护 y = wx + b 的线性模型，参数 w（斜率）和 b（截距）以在线方式逐样本更新。每接收一个 (x, y) 样本对，使用梯度下降公式 w = w - lr * dw, b = b - lr * db 更新参数，其中 dw 和 db 分别为损失函数（均方误差）对 w 和 b 的偏导数。学习率 lr 在初始化时由调用方指定。
2. 实现点预测：给定一个 x 值，使用当前模型参数 w 和 b 返回预测值 y_pred = w * x + b。
3. 实现 R² 拟合优度计算：基于所有已接收的样本数据，计算模型的 R²（决定系数）。R² = 1 - SS_res / SS_tot，其中 SS_res 为残差平方和，SS_tot 为总平方和。需在在线更新过程中维护必要的中间统计量（如样本数、x 和 y 的和与平方和、xy 乘积和），避免存储全部历史数据。
4. 代码实现需要放在 src/solocoder_py/linear_regression/ 包下，然后为该模块编写完整的单元测试，覆盖正常流程（完美线性数据 y=2x+3 的模型参数趋近理论值、点预测结果与手动计算一致、R² 在拟合良好数据上接近 1、R² 在随机噪声数据上接近 0、多次更新后参数的收敛趋势）、边界条件（仅一个样本时的 R² 处理、学习率为极小值时的更新行为、零学习率的处理、x 全相同时的回归退化情况）和异常分支（学习率为负数的构造拒绝、输入 NaN 或 Inf 样本数据的处理、未训练时调用 predict 的行为、未训练时查询 R² 的行为、样本数为零时的处理）。测试代码放在 tests/linear_regression/ 下。
5. 在该模块包内编写 README.md，说明模块功能、核心类的职责、梯度下降更新公式与推导、R² 计算公式与在线统计量维护、以及使用示例。
