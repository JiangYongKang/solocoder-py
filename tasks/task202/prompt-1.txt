在当前 Python 项目中实现一个先进先出（FIFO）队列数据结构，使用 Python 内置的数据结构作为底层存储。
1. 实现队列的基本操作：将元素加入队尾、从队首取出元素并返回、查看队首元素但不移除。
2. 实现队列的查询操作：判断队列是否为空、获取当前队列中的元素数量。
3. 边界处理：当从空队列取出元素或查看队首时，应抛出明确的异常而不是返回 None 或产生未定义行为。
4. 代码实现放在 src/solocoder_py/basic_queue/ 包下，为该模块编写完整的单元测试，覆盖正常流程（按入队顺序出队、查看队首、判空、获取大小）和边界条件（空队取出抛异常、空队查看队首抛异常、大量元素入队后逐一取出、入队出队交替执行后元素顺序正确）。测试代码放在 tests/basic_queue/ 下。
