在当前项目中实现一个 JSONPath 查询域功能模块，使用内存数据结构模拟数据源。
1. 实现字段选择：支持用点号或方括号语法访问 JSON 对象的指定字段，返回对应字段的值，字段不存在时返回 None 或空列表。
2. 支持数组下标选择：支持按数字下标访问 JSON 数组中的指定元素，下标从零开始计数，下标越界时不做异常抛出而是返回空结果。
3. 支持通配符选择：使用星号通配符选择对象的所有子字段或数组的所有元素，返回匹配到的多个结果组成的列表。
4. 支持递归下降搜索：使用双点号语法在整个 JSON 文档树中递归搜索指定字段名，不限嵌套深度，返回所有匹配到的结果。
5. 支持多结果集合返回：查询结果以列表形式返回，若查询路径匹配到多个值则全部包含在结果列表中，保持文档中的原始顺序。
6. 代码实现需要放在 src/solocoder_py/jsonpath/ 包下，然后为该模块编写完整的单元测试，覆盖正常流程、边界条件（如查询不存在的字段、空数组通配、递归搜索无匹配结果）和异常分支（如非法路径语法、在非数组元素上使用下标）。测试代码放在 tests/jsonpath/ 下。
7. 在该模块包内编写 README.md，说明模块功能、核心类的职责、支持的 JSONPath 语法规范、查询结果类型约定以及使用示例。
