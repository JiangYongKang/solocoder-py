在当前 Python 项目中实现一个基于数组的哈希表，使用链地址法（拉链法）解决哈希冲突，用 Python 内置列表和链表模拟。
1. 实现哈希表的基本操作：支持插入键值对、按键查找返回值、按键删除键值对。
2. 实现动态扩容：当哈希表中元素数量与槽位数比值超过设定的负载因子阈值时，自动扩容槽位数组并重新哈希所有已有键值对，扩容过程用户无感知。
3. 实现哈希表的查询：判断是否包含指定键、获取当前键值对数量、获取当前负载因子。
4. 边界处理：查找或删除不存在的键应返回未找到的明确结果，插入已存在的键应更新值而非新增。
5. 代码实现放在 src/solocoder_py/hashtable_chaining/ 包下，为该模块编写完整的单元测试，覆盖正常流程（插入查找删除、更新已存在的键、判键存在、获取大小和负载因子、动态扩容触发后数据完整）和边界条件（空表操作、删除不存在的键、扩容过程中查找仍在工作、负载因子刚好在阈值临界值）。测试代码放在 tests/hashtable_chaining/ 下。
6. 在该模块包内编写 README.md，说明模块功能、核心类的职责、链地址法原理、扩容策略以及使用示例。
