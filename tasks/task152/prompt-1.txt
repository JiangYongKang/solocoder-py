在当前 Python 项目中实现 Unicode 分词器域，使用内存数据结构处理文本输入。
1. 实现 CJK 单字切分：对中日韩统一表意文字（CJK Unified Ideographs，Unicode 基本区间 U+4E00 至 U+9FFF 及扩展区 A 至 G）按单字为单位切分，每个汉字单独作为一个 Token。CJK 字符不依赖空白或标点分隔。
2. 实现标点处理：对各种标点符号进行独立 Token 化处理，中文标点（全角逗号、句号、引号、书名号等）和西文标点（半角逗号、句点、引号等）均独立成 Token。连续的英文字母或数字字符按空白或标点边界切分为词级 Token，而非逐字切分。
3. 实现按语言激活不同规则集：根据输入文本检测到的 Unicode 脚本（拉丁字母、西里尔字母、阿拉伯字母等），自动激活对应的分词规则集。拉丁字母文本按空格和标点分词，CJK 文本按单字切分，其他连续书写文字（如泰文）采用字典辅助分词或降级为空格切分。
4. 代码实现需要放在 src/solocoder_py/tokenizer/ 包下，然后为该模块编写完整的单元测试，覆盖正常流程（中英混合文本分词、纯中文单字切分、纯英文按空格分词）、边界条件（空字符串返回空列表、仅标点文本、极长连续 CJK 字符）和异常分支（Emoji 等代理对字符不破坏切分逻辑、未知脚本的降级处理、连续多个标点作为独立 Token 而非合并）。测试代码放在 tests/tokenizer/ 下。
5. 在该模块包内编写 README.md，说明模块功能、核心类的职责、分词规则集与 Unicode 脚本检测机制（可用文本描述）以及使用示例。
